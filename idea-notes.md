## Home Medicine Cabinet - MVP

### Main problem
With a large number of medications, it's impossible to remember all the medications in your home medicine cabinet, their quantities, and expiration dates. This can lead to shortages or purchasing medications already in your cabinet.

### The smallest set of functionalities
- A simple user account system for storing medication data (username + password)
- Adding medications with their expiration dates (medications can only be added from the list of medications approved in Poland)
- Viewing, editing, and deleting medications
- The user can set the time (number of days) when to receive notifications that medications are close to expiration (max 90, minimum 7 days)
- Dashboard, which contains information: number of all medicines, valid medicines, medicines nearing their expiration date, and expired medicines
- Takes a search tern and finds medications whose name, or active ingredients match the term

### What is NOT included in the MVP
- Checking medications based on a newly issued prescription (if available in the home medicine cabinet)
- Adding medications using other formats (photos, PRD, etc)
- Creating profiles of household members and prescribing medications to them
- Medications dosage monitoring
- Integration with other educational platforms
- Mobile applications (web-only initially)

### Success Criteria
- Searching for drugs by name or active substance returns correct results (95% success)
- Medications are correctly identified as nearing expiration based on user settings
